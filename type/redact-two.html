Usage

Simply apply a class of redacted along with a class of r1 thru r5.

Example: 
The following is <span class="redacted r1">redacted</span>

Result:

The following is redacted

Random Redaction

In lieu of adding classes r1 thru r5, add data-random to the span.

Example: 
Random <span class="redacted" data-random>redaction</span>

Result:

Random redaction
